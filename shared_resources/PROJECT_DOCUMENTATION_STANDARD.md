# ⚖️ **무조건 적용 규칙 (MANDATORY RULES)**

## 🎯 **규칙 1: 필수 문서화 체크리스트**

**모든 프로젝트는 다음 5개 문서를 필수로 작성해야 합니다:**

```
☐ AI 팀 기술자료 문서화       
☐ 스펙 및 아키텍처 가이드 작성
☐ 코딩방법 매뉴얼 작성
☐ 전체 코드 공유폴더 저장
☐ 사용법 가이드 통합
```

**⚠️ 중요: 이 5개 항목 중 하나라도 누락되면 프로젝트 완료로 인정하지 않습니다.**

# 📁 **규칙 2: 표준 디렉토리 구조**

## 🏗️ **필수 디렉토리 구조**

모든 프로젝트는 다음 구조를 정확히 따라야 합니다:

```
/project_root/
├── shared_database/                    # 데이터베이스 파일
│   └── [project_name].db
├── shared_resources/                   # 📋 필수 문서 저장소
│   ├── [PROJECT_NAME]_DEVELOPMENT_GUIDE.md           # ☐ 기술자료 문서화
│   ├── SYSTEM_SPECS_ARCHITECTURE_GUIDE.md            # ☐ 스펙 및 아키텍처  
│   ├── CODING_METHODS_MANUAL.md                      # ☐ 코딩방법 매뉴얼
│   ├── CODE_COLLECTION.md                            # ☐ 코드 모음집
│   ├── INTEGRATED_USER_GUIDE.md                      # ☐ 사용법 가이드 통합
│   ├── complete_source_codes/                        # 💻 소스코드 보관소
│   │   ├── README.md                                # 코드 설명서
│   │   └── *.js, *.html, *.css, *.sql, etc.        # 모든 소스파일
│   ├── master_[system_name]/                         # 주요 시스템별 폴더
│   └── announcements/                                # 공지사항
├── simple_backups/                     # 백업 저장소
├── logs/                              # 로그 파일
└── 핵심 시스템 파일들 (프로젝트 루트)
```
