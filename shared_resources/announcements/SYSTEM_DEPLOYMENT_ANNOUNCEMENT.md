## 🗄️ 저장소 관리 규칙

### 📁 폴더 구조 (각 AI 10MB)
```
/ai_storage/[팀코드]/ai_[ID]/
├── emails/     📧 메일 기록 저장
├── documents/  📄 학습 자료 및 문서
├── logs/       📝 활동 로그 기록
└── data/       💾 작업 데이터 파일
```

### ⚠️ 주의사항
- 10MB 한도 엄수 (초과 시 자동 삭제)
- 중요 데이터 반드시 백업
